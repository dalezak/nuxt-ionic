# Nuxt Layer Ionic

## Packages

Install packages

```bash
npm install
```

Reinstall packages

```bash
npm run reinstall
```

Update packages
```bash
npm update
```

## Development

Local environment on `http://localhost:3000`

```bash
npm run local -- --open
```

Production environment

```bash
npm run prod -- --open
```

## Production

Production build

```bash
npm run build
```

## Clean

Clean cache

```bash
npm run clean
```
